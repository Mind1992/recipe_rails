# Recipes Rails
## About
This project is for [Epicodus'](http://www.epicodus.com/) [Rails basics assessment](http://www.learnhowtoprogram.com/lessons/rails-basics-assessment). Check [live version](http://recipesapp.herokuapp.com/) on heroku.
This project allows users to add new recipes with description and also add tags to it. User also can edit, delete and list recipes and tags.

## Setup
Clone this repository or download the .zip. In Terminal open up the folder recipes_rails and run $rails server. Then go to your browser and go to this link http://0.0.0.0:3000/.
